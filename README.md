# NumPy Arithmetic Operations and Comparisons

This project demonstrates NumPy arithmetic operations, vectorized mathematical functions, comparison operators, and conditional array modification in Python.

## Topics Covered

### Scalar Arithmetic

Perform mathematical operations on every element of an array using a single value.

```python
arr + 2
arr - 2
arr * 2
arr / 2
arr ** 2
```

### Vectorized Mathematical Functions

Apply mathematical functions to entire arrays without loops.

```python
np.sqrt(arr)
np.round(arr2)
np.floor(arr2)
np.ceil(arr2)
```

### Element-wise Arithmetic

Perform operations between two arrays of the same size.

```python
arr3 + arr4
arr3 - arr4
arr3 * arr4
arr3 / arr4
```

### Comparison Operators

Create Boolean arrays using comparison conditions.

```python
marks >= 50
marks < 50
```

### Boolean Indexing

Modify array values based on conditions.

```python
marks[marks < 40] = 0
```
